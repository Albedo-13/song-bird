# <p align="center">Song Bird</p>

## About the project ℹ️
Songbird is a quiz application that allows the user to test their skills in identifying birds by their singing. It's harder than it looks!

👉 Live Demo: [Live Demo](https://song-bird-albedo.netlify.app/)
